# swiper-lazy-render
延迟渲染大型swiper列表
 
* ### 过长swiper列表按需渲染
* ### 解决问题：swiper列表过长导致渲染耗时卡顿，滚动不流畅
* ### 本指令目标：只渲染当前激活下标周围SWIPER_SLIDE_SIZE个内容，设置为奇数个，个数越小渲染越快，滑动效果越流畅
